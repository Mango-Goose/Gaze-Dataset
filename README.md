# Gaze Validation
## A new dataset for model evaluation

This dataset was created with the intention of validating model 3D prediction results and also catching examples of overfitting with synthetic data

![images](./read_me_images/Dataset_img.png)

## Annotations

The dataset contains extensive annotations on the following topics: 
 - Image path
 - Image width and height
 - Camera position and rotation
 - Person position and rotation
 - 2D gaze point
 - 3D gaze point
 - Gaze object
 - Whether the object is occluded or not
 - Table position and rotation
 - Head bounding box (2D)
 - Head position (3D)
 - List of 2D object bounding boxes
 - List of object’s 3D positions
 - List of object’s labels
 - Package Number

The intention is for this dataset to be of use for research purposes when completing 2D and 3D gaze following and object detection tasks
